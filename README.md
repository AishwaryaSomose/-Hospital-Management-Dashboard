🏥 Hospital Management Dashboard (Power BI)
 Overview

This project presents an interactive Hospital Management Dashboard built using Power BI to analyze hospital operations, financial performance, and patient trends. It helps stakeholders make data-driven decisions by providing clear and actionable insights.

-Objectives
Analyze hospital revenue and payment trends
Understand patient visit patterns
Evaluate insurance provider contributions
Track performance using KPIs
Enable interactive data exploration
📊 Key Insights
💰 Total revenue analysis across payment methods (Cash, Credit Card, Insurance)
📌 Payment status tracking (Paid, Pending, Failed)
🏥 Patient visit reasons (Checkup, Consultation, Emergency, Follow-up, Therapy)
🛡️ Insurance provider performance (HealthIndia, MedCare Plus, PulseSecure, WellnessCorp)
👥 Gender-based analysis of patients
🎯 KPI tracking for performance vs targets
🧩 Features
Interactive filters (Payment Method, Status, Specialization)
Dynamic visuals for deep analysis
Clean and user-friendly UI design
Drill-down capabilities for better insights
🛠️ Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Data Modeling
Data Cleaning & Transformation (Power Query)
📂 Dataset

The dataset includes hospital-related data such as:

Patient details
Billing information
Payment methods & status
Insurance providers
Visit reasons & specialization
